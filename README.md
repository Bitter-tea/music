# 说明

仿造 [前端基础必会教程-4 个小时带你快速入门 vue](https://www.bilibili.com/video/BV12J411m7MG?p=1)<br>
[网易云音乐 API](https://autumnfish.cn/)<br>

# 功能

1. 音乐搜索
2. 音乐播放
3. mv 播放
4. 热门评论
5. 封面动画
6. 换背景

# 技术栈

> HTML5+CSS3+javaScript+vue2+store+iView
